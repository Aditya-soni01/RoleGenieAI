\# RoleGenie Codex Instructions



You are working on RoleGenie, a full-stack AI resume optimization SaaS.



Backend:

\- Python FastAPI

\- SQLAlchemy 2

\- Alembic

\- Pydantic v2

\- JWT auth

\- AI providers: OpenAI, Anthropic, OpenRouter

\- Resume upload, optimization, PDF/DOCX export

\- Render deployment



Frontend:

\- React 18

\- TypeScript

\- Vite

\- Tailwind CSS

\- Zustand

\- TanStack Query

\- Axios

\- Hostinger deployment



Rules:

\- Do not break existing auth flow.

\- Do not change API routes without updating frontend calls.

\- Keep backend routes under /api.

\- Keep AI provider logic configurable through env vars.

\- Before coding, explain affected files.

\- After coding, provide test commands.

\- Avoid over-engineering.

\- Prefer small safe changes.

\- Do not edit deployment configs unless requested.
